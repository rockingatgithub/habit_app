# habit_app
A daily habit tracker app
To run this file.
Steps:_
1. clone this repo
2. in terminal run "npm install"
3. run npm start
4. Open 'localhost:8000'
Note:- 8000 is port on which server is running make sure port is free.
       you can run 'netstat -plant' to view active port.
